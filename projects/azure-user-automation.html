Azure User Automation Lab

Microsoft Azure | User Management | Automation | Cloud Administration

Project Overview

This project demonstrates how to create and manage cloud users in Microsoft Azure. The goal was to practice real-world identity administration tasks used by IT support, cloud administrators, and cybersecurity teams.

Tools Used

  • Microsoft Azure Portal
  • Microsoft Entra ID
  • PowerShell
  • GitHub

Skills Demonstrated

  • User account creation
  • Group management
  • Cloud identity administration
  • Basic automation
  • Technical documentation

What I Learned

I learned how cloud-based identity management works and how administrators create, organize, and manage users in Microsoft Azure environments.

Enterprise Active Directory Administration Lab

This project demonstrates enterprise Active Directory administration using Windows Server 2022 hosted in Microsoft Azure. The goal was to build a realistic corporate environment that simulates how IT teams manage users, groups, organizational units, permissions, and policies at scale.

Project Objectives

  • Deploy Windows Server 2022 in Microsoft Azure
  • Install and configure Active Directory Domain Services
  • Create a corporate OU structure
  • Implement Role-Based Access Control (RBAC)
  • Create and manage security groups
  • Automate user provisioning with PowerShell
  • Apply Group Policy Objects (GPOs)

Environment

  • Microsoft Azure
  • Windows Server 2022
  • Active Directory Domain Services
  • Group Policy Management
  • PowerShell

Organizational Unit Design

Branch_01
│
├── Users
│   ├── HR
│   ├── IT
│   └── Finance
│
├── Devices
├── DisabledUsers
└── RetiredDevices

Branch_02
│
├── Users
├── Devices
├── DisabledUsers
└── RetiredDevices

The OU hierarchy was designed to mirror a real enterprise environment. This structure allows administrative delegation, scalable management, and targeted Group Policy deployment.

Role-Based Access Control (RBAC)

Security groups were created to manage permissions using RBAC principles. The following groups were implemented:

  • IT_Admin
  • HR_Staff
  • Finance_Users
  • Workstation_Admins

PowerShell Automation

A PowerShell automation script was developed to create Organizational Units, Security Groups, and Active Directory user accounts automatically from a CSV file. This reduced manual administrative effort and simulated enterprise onboarding processes.

Group Policy Implementation

A workstation administration Group Policy Object was created and linked to the Branch_01 Devices OU. This demonstrates centralized management of workstation permissions and administrative access through Group Policy.

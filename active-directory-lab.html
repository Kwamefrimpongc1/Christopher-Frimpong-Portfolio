Enterprise Active Directory Administration Lab

Built a Windows Server 2022 Active Directory environment in Microsoft Azure. This project simulated a real enterprise environment with users, groups, organizational units, PowerShell automation, RBAC, and Group Policy Management.

Technologies Used

Microsoft Azure

Windows Server 2022

Active Directory Domain Services

Group Policy Management

PowerShell

What I Built

  • Deployed a Windows Server 2022 Domain Controller in Azure
  • Created the corp.local Active Directory domain
  • Designed an enterprise OU structure with Branch_01 and Branch_02
  • Created IT, HR, and Finance departmental OUs
  • Created security groups for RBAC
  • Automated user provisioning using PowerShell and CSV imports
  • Configured and linked Group Policy Objects
